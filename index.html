Orbe.js

This project is a JavaScript implementation of ORBE, an Orbital Integrator originally written in FORTRAN.

Getting Started

Installing Orbe.js is very simple. All you have to do is copying the Orbe.js file to your working folder. 
        
<div id="orbe"></div>

<script type="text/javascript" src="./Orbe.js"></script>
<script type="text/javascript">
    // Orbe.load(JSON_PATH, ELEMENT_ID);
    Orbe.load('orbedata.json', 'orbe');
</script>
