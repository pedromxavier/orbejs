Orbe.js

This project is a JavaScript implementation of ORBE, an Orbital Integrator originally written in FORTRAN.

Getting Started

Installation

Installing Orbe.js is very simple. All you have to do is copying the Orbe.js file to your working folder. It is available in the /src folder at the project's repository.

        
<div id="orbe"></div>

<script type="text/javascript" src="./Orbe.js"></script>
<script type="text/javascript">
    Orbe.load(path='./orbeini.json', id='orbe');
</script>
