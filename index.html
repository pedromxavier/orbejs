Orbe.js

This project is a JavaScript implementation of ORBE, an Orbital Integrator originally written in FORTRAN.

Getting Started

Installation

Installing Orbe.js is very simple. All you have to do is copying the Orbe.js file to your working folder. It is available in the /src folder at the project's repository.

Running

When including the Orbe.js script, the Orbe class becomes available. There are two main ways to run a simulation:

Loading JSON file

        
<div id="orbe"></div>

<script type="text/javascript" src="./Orbe.js"></script>
<script type="text/javascript">
    Orbe.load(path='./orbeini.json', id='orbe');
</script>

Data Patterns

  • \(a \in [0, \infty)\): Semimajor axis of the ellipse in astronomical units.
  • \(e \in [0, 1]\): Eccentricity of the ellipse. \(0\) for a circle.
  • \(i \in [0^\circ, 180^\circ]\): Inclination of the orbit in degrees. \(0^\circ\) if it's coincident with plane \(xy\) or ecliptic, \(90^\circ\) perpendicular and \(180^\circ\) retrograde motion.
  • \(W \in [0^\circ, 360^\circ]\): Longitude of the ascending node in degrees. \(0^\circ\) corresponds to Aries or \(x\)-axis, usually written as \(\Omega\).
  • \(w \in [0^\circ, 360^\circ]\): Argument of the perihelion in degrees. \(0^\circ\) corresponds to the direction of the node, usually written as \(\omega\).
  • \(M \in [0^\circ, 360^\circ]\): Mean anomaly in degrees. \(0^\circ\) corresponds to perihelion, \(180^\circ\) to aphelion.
  • \(m \in [0, \infty)\): Mass in solar masses.
